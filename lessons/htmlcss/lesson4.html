CSS: Basics

CSS is like painting your house. HTML gives the structure, CSS makes it look pretty with colors, sizes, and positions.

Example:

<h1 style="color:blue; text-align:center;">Hello World</h1>
<p style="font-size:18px;">This is my first styled paragraph.</p>

Try It Yourself:

Create a heading with red color and a paragraph aligned to the center.
